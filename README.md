# USA Chemistry & Pharmacy Grad Finder

This is an interactive page to explore **chemistry and pharmacy graduate programs** in the USA, including scores, fees, and funding info.

## Features
- Filter by **state**, **tier** (low/mid/high/Ivy), and **area** (pharmacy/chemistry).
- See **required score hints** (TOEFL/IELTS/GRE).
- Filter for **funded vs self‑fund** options.

## How to use
1. Open the page (e.g., via GitHub Pages URL).
2. Use the filters to find programs that match your background and budget.
3. Click **Open** links to go directly to university application pages.

This site is designed for international students from countries like Bangladesh who want to see many universities in one place.
